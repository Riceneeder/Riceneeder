| 狗脊     | 蚌壳蕨科植物金毛狗脊                   | 干燥根茎           | 药材：表面深棕色，残留金黄色绒毛；饮片：边缘不齐，偶有金毛残留，切面浅棕色。边缘有棕黄色隆起的木质环纹或条纹 |

| 细辛     | 马兜铃科植物北细辛、汉城细辛、华细辛   | 干燥根和根茎       | 根细长，有须根和须根痕，黄白色或白色，气辛香，味辛辣、舌麻   |
| 大黄     | 蓼科植物掌叶大黄、唐古特大黄、药用大黄 | 干燥根和根茎       | 表面黄棕色至红棕色，有的可见类白色网状纹理及星点散落；断面淡红棕色或黄棕色，根茎髓部宽广，有星点环列或散落；根部具有放射状纹理，形成层环明显，无星点；气清香，味苦；草酸钙簇晶大而多，有沙粒感 |
| 何首乌   | 蓼科植物何首乌多                       | 干燥块根           | 外形与红薯相似；不宜折断，粉性；皮部有4-11个类圆形异形维管束环列，形成云锦状花纹，气微 |

| 川牛膝   | 苋科植物川牛膝                         | 干燥根             | 表面黄棕色或灰褐色，质地坚韧，断面浅黄色或棕黄色，维管束点状，排列成数轮同心环（比牛膝的环明显） |
| 商陆     | 商陆科植物商陆、垂序商陆               | 干燥根             | 外皮灰黄色或灰棕色，横切片弯曲不平，切面浅黄棕色或黄白色，木部隆起，形成数个突起多同心环轮，习称“罗盘纹” |
| 太子参   | 石竹科植物孩儿参                       | 干燥根             | 细长纺捶形或细长条形，黄白色，较光滑；质硬脆，断面平坦，淡黄白色。角质样，粉性；味道与健胃消食片相似 |
| 威灵仙   | 毛茛科植物威灵仙                       | 干燥根和根茎       | 外形与细辛相似，外表黄黑色，质硬脆，易折断                   |
| 川乌     | 毛茛科植物乌头                         | 干燥母根           | 表面棕褐色或灰棕色，皱缩，有小瘤状侧根及子根脱离后的痕迹；质坚实，断面类白色或浅灰黄色，形成层环纹呈多角形；气微，昧辛辣、麻舌；形似乌鸦头 |
| 附子     | 毛茛科植物乌头                         | 子根               | 盐附子：具有盐霜；黑顺片：外皮黑褐色，切面暗黄色，油润有光泽；白附片：无外皮，黄白色，半透明 |
| 白头翁   | 毛茛科植物白头翁                       | 干燥根             | 表面黄棕色或褐棕色；根头部稍微膨大，有白色绒毛，断面皮部黄白色或淡黄棕色 |
| 白芍     | 毛茛科植物芍药                         | 干燥根             | 表面类白色或淡红棕色，光洁或有纵皱纹，不宜折断，断面较平坦，类白色或微带棕红色。形成层环明显，射线放射状，气微 |
| 赤芍     | 毛茛科植物芍药                         | 干燥根             | 表面棕褐色或棕红色，断面粉白色或粉红色                       |

| 防己     | 防己科植物粉防己                       | 干燥根             | 形似大肠，断面平坦，灰白色，粉性，有排列稀疏多放射状纹理，习称“车轮纹” |

| 苦参     | 豆科植物苦参                           | 干燥根             | 表面灰棕色或棕黄色，栓皮多破裂后反卷，易剥落不易折断，断面纤维性，黄白色，味极苦 |
| 葛根     | 豆科植物野葛                           | 干燥根             | 纵切多长方形厚片或小方块，外皮淡棕色，粗糙，切面黄白色至淡黄棕色，有的纹理明显。纤维性强 |
